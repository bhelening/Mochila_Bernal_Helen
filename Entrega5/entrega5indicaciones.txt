Abrir la terminal en la máquina virtual Ubuntu
Para generar una nueva clave SSH, ejecutar: ssh-keygen -t rsa -b 4096 -C bhelenbv@gmail.com 
dejé en blanco la contraseña
utilizar el archivo publico (.pub) para la autenticaicon con git:
	En Settings>SSH and GPG key>New SSH key
	Copiar del archivo .pub y pegar en "Key" la clave 
	Add
Clonar mi repositorio de GitHub 
En la página del repositorio en GitHub, Abrir "Code" y seleccioné la opción "SSH" para obtener la URL SSH del repositorio.
Ejecuté git clone y  mi url del repositorio SSH<URL_DEL_REPOSITORIO_SSH>, donde <URL_DEL_REPOSITORIO_SSH>
Para subir el archivo se siguen los pasos de siempre: git add, el commit y el push.
